1. git add .
   (Lệnh này báo cho Git biết: "Tôi muốn lấy tất cả những gì mới làm/sửa để chuẩn bị đóng gói").

2. Đóng gói và ghi chú
   Bash

git commit -m "Ten ghi chu cua ban"
(Thay dòng trong ngoặc kép bằng nội dung bạn muốn, ví dụ: "Lam xong bai 15", "Sua loi CSS bai 10"... Lệnh này giúp đóng gói code lại thành một phiên bản).

3. Đẩy lên mạng
   Bash

git push
(Lệnh này chuyển cái gói vừa đóng lên GitHub. Chạy xong thấy dòng "100% done" là thành công).
